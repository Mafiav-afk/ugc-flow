# UGC Flow 视频讲解说明与录制脚本

建议成片：6-8 分钟，16:9 横屏，1080p。  
目标：让新用户完成安装、API 配置、商品图上传和第一条 Grok 带货视频。

## 录制准备

- 隐藏 API Key，录屏时不要展示真实密钥。
- 准备一张无隐私信息的商品图。
- 浏览器缩放保持 100%，关闭无关标签页和通知。
- 示例模型使用 `grok-video-3`，先选择 10 秒与 720P。

## 分镜与旁白

### 00:00-00:25 开场

画面：展示 UGC Flow 工作台和完成的视频预览。  
旁白：“这是一套本地运行的带货视频工作流。Codex 或 Claude 负责商品分析和脚本，媒体 API 只负责图片与视频生成。”

### 00:25-01:10 安装

画面：打开 DMG，双击安装器，选择 Codex/Claude。  
旁白：“下载 Mac 安装包后打开安装器，可一键部署到 Codex、Claude Code，或者同时部署。安装完成后会自动启动本地服务。”

### 01:10-02:00 填写商品信息

画面：输入产品名、卖点、人群，上传商品图。  
旁白：“卖点要真实、具体，建议填写三到六个。图生视频必须上传一张清晰商品图，系统会锁定包装和外观。”

### 02:00-03:20 配置 API

画面：进入“智能体与 API”，关闭演示模式。Key 区域打码。  
旁白：“这里填写视频 API 根地址与密钥。以 ToAPIs 为例，选择普通兼容 JSON，模型填写 grok-video-3。首次验证建议使用十秒、九比十六和七百二十 P。”

画面：点击“智能识别并测试”。  
旁白：“测试成功后再保存。模型名称必须来自供应商文档或模型列表，不能把不同供应商的别名混在一起。”

### 03:20-04:50 一键生成

画面：点击“一键生成带货视频”，依次展示六个阶段。  
旁白：“工作流会分析商品、生成八镜头策略、准备参考图、锁定产品，再提交异步视频任务。右侧会先看到图片，然后实时看到视频进度。”

### 04:50-05:40 预览与下载

画面：播放视频，展示复制提示词、导出项目、下载视频。  
旁白：“任务完成后可以直接播放、下载视频、复制提示词或导出项目。供应商视频链接可能只有二十四小时有效，建议及时保存。”

### 05:40-06:40 故障处理

画面：展示 API 配置与运行日志，不显示密钥。  
旁白：“遇到四零四时检查 Base URL；提示缺少参考图时检查公网图片 URL；提示 duration 类型错误时确认它是数字；提示 pricing rule 时更换已开通的时长或分辨率。”

### 06:40-07:00 收尾

画面：回到成功成片。  
旁白：“文本消耗使用自己的 Codex 或 Claude 额度，媒体费用由所填 API 产生。先用低成本规格验证，再制作正式成片。”

## 画面标注建议

- API Key 区域全程遮罩。
- 关键按钮使用红色圆框或局部放大。
- 阶段进度用字幕标注“图片已回传”“视频任务已接受”“成片可下载”。
- 错误示例只显示错误类型，不显示 request ID、Key 或私人商品信息。

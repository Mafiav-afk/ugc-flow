# UGC Flow 软件操作说明

版本：1.3.0（Mac Apple Silicon）  
适用对象：商家、短视频运营、UGC 创作者

## 1. 安装与启动

1. 双击 `UGC-Flow-Mac-1.3.0-arm64.dmg`。
2. 打开“UGC Flow 安装器”，选择部署到 Codex、Claude Code 或两者。
3. 安装完成后访问 `http://127.0.0.1:18787/`。
4. 首次启动进入“智能体与 API”，确认 Codex/Claude 已连接。

> API Key 只保存在当前页面会话。重新安装、关闭标签页或清理浏览器状态后需要重新填写。

## 2. 最快生成流程

1. 填写产品名称、3-6 个真实卖点和目标人群。
2. 上传 1 张清晰商品图；图生视频模型必须有参考图。
3. 选择爆款格式，或保持“自动匹配最佳格式”。
4. 打开“智能体与 API”，关闭演示模式。
5. 输入媒体 API Base URL 与 Key，选择视频模型、时长、分辨率和比例。
6. 点击“智能识别并测试”，确认连接正常。
7. 保存配置，点击“一键生成带货视频”。
8. 在右侧依次预览图片、视频，完成后点击“下载视频”。

## 3. 图片来源

- Codex 生图：Codex 生成图片提示词；环境能返回下载 URL 时直接使用，否则自动切换图片 API。
- 异步图片 API：提交图片任务并轮询公网图片 URL。
- 上传参考图：直接使用商品图；若视频接口要求公网 URL，系统尝试供应商上传接口。

## 4. 推荐的 ToAPIs Grok 配置

- Base URL：`https://toapis.com/v1`
- 协议：普通兼容 JSON
- 模型：`grok-video-3`
- 时长：先用 10 秒；15 秒需要当前渠道已配置对应计费规则
- 比例：`9:16` 或 `16:9`
- 分辨率：`720P` 或 `1080P`
- 图片来源：上传参考图或可返回公网 URL 的图片 API

## 5. 任务阶段

1. 智能体分析：Codex/Claude 使用自身额度分析商品。
2. 镜头策略：生成 8 镜头脚本。
3. 图片生成：生成或接收图片并回传预览。
4. 产品锁定：固定包装、颜色、标签与外观。
5. 视频生成：提交异步视频任务并轮询进度。
6. 成片：播放器显示视频，可复制提示词、导出项目和下载视频。

## 6. 常见问题

- `404 Not Found`：Base URL 或端点不匹配。优先填写 API 根地址并运行智能识别。
- `one reference image is required`：模型仅支持图生视频，必须上传图片并确保其公网可访问。
- `duration ... type int`：时长必须传数字，不是字符串。
- `pricing rule not matched`：当前渠道未配置该模型/时长/分辨率价格档，改用已开通规格。
- 一直在智能体分析：Codex/Claude 首次运行可能较慢；超时后系统会切换本地商家模板继续。
- 视频无声音：只有明确支持音频的模型才保证声音；界面会标记“有声”。

## 7. 安全与合规

- 不要在截图、群聊或文档中公开 API Key。
- 商品功效、医疗、美白、抗衰等表述必须有真实材料支撑。
- 视频 URL 可能有有效期，生成成功后请及时下载归档。

# Hook Mechanisms

- Discovery: “I have done [specific task] for [specific time], and only now found this?”
- Disbelief: show the result first, then ask what caused it.
- Pain specificity: name the exact moment, place, and friction instead of a broad problem.
- Identity: call out a lived behavior the audience recognizes in themselves.
- Comparison: show two methods under the same visible condition.
- Confession: admit the old workaround before revealing the product.
- Challenge: set a measurable test the viewer can watch finish.
- Social moment: let a partner, parent, coworker, or pet reaction become proof.

Rewrite every hook with the actual product, user, scenario, and evidence. Do not copy these lines verbatim into final ads.
